# ИГРЫ РАЗУМА

Консольное приложение, представляющее из себя набор из пяти простых математических игр. Они построены по принципу популярных мобильных приложений для прокачки мозга.

## Минимальные требования

IQ не ниже 80.

## Установка и запуск

Игры устанавливаются командой make install. Запускаются отдельно командами: brain-calc, brain-gcd, brain-even, brain-progression, brain-prime.
